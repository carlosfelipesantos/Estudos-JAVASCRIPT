Estudos em JavaScript (com Node.js e Express)
Este repositório reúne meus exercícios e projetos desenvolvidos durante meus estudos em JavaScript, registrando minha evolução na linguagem. 
O objetivo é praticar de forma constante, explorar desde conceitos básicos até recursos mais avançados e criar um histórico de aprendizado no GitHub.
Aqui você encontrará códigos abordando fundamentos como variáveis, operadores, funções, manipulação do DOM e eventos, além de tópicos intermediários como programação assíncrona e consumo de APIs.
